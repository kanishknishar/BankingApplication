# BankingApplication
This exercise to test my knowledge of ArrayLists was an assignment on Tim Buchalka's Java Udemy course. You create a banking application which would have a bank class which hold an ArrayList of its branches. The branch class would have a string name and itself would have an ArrayList of customers. Finally, the customer class would have a string name and an ArrayList of transactions.  The program runs entirely by itself without the need for manually inputting data into the code itself.

The version wrote by the instructor was more primitive than mine. My version of the program tries to use much cleaner code.
